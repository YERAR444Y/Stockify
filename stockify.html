Centro de Control Stockify

Guía de usuario para el sistema de gestión de CASS Colombia Recubrimientos S.A.S.

1
Lógica del Sistema

El código utiliza un Array de Objetos en JavaScript para simular una base de datos en tiempo real. Cada producto cuenta con:

  • SKU: Identificador único (CAS-XXX).
  • Lote: Seguimiento de producción.
  • Punto de Reorden: Cantidad mínima antes de disparar alertas.
Estructura: {sku, nombre, stock, reorden...}

2
Gestión de Inventario

En la pestaña 📦 Inventario puedes registrar nuevos químicos. El sistema valida que el SKU sea único y lo añade automáticamente al catálogo maestro.

Tip: Usa la barra de búsqueda para filtrar por lote o nombre instantáneamente.

3
Entradas y Salidas

Para alterar el stock, ve a 🔄 Entradas / Salidas:

  1. Selecciona el Modo Operación (Verde para entrada, Rojo para salida).
  2. Usa los botones +1 / +10 para ajustar cantidades.
  3. El sistema bloquea salidas si el stock es insuficiente.

4
Inteligencia y Alertas

Dashboard: Visualiza tendencias mediante Chart.js. Los gráficos se actualizan con cada movimiento.

Asistente CASS: Haz clic en el globo de texto azul abajo a la derecha para consultar el stock por voz o texto sin navegar por las tablas.

⚠️ Reglas de Uso Importantes

1. Los datos se mantienen mientras la pestaña esté abierta (Persistencia en Sesión).

2. Las fechas de vencimiento deben seguir el formato AAAA-MM para el correcto filtrado del Chatbot.

3. Revise la sección 🔔 Alertas diariamente para evitar quiebres de stock en producción.
